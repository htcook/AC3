# WebGoat Vulnerability Categories (for Ground Truth)

Based on OWASP WebGoat 2023.8+ lesson structure:

## Lesson Categories
1. **General** - HTTP Basics, Developer Tools
2. **Injection** 
   - SQL Injection (Intro, Advanced, Mitigations)
   - Path Traversal
   - XXE (XML External Entity)
3. **Broken Authentication**
   - Authentication Bypasses
   - JWT (JSON Web Token) attacks
   - Password Reset
   - Secure Passwords
4. **Sensitive Data Exposure**
   - Insecure Login (cleartext credentials)
5. **XML External Entities (XXE)**
   - XXE attacks
6. **Broken Access Control**
   - Insecure Direct Object References
   - Missing Function Level Access Control
7. **Cross-Site Scripting (XSS)**
   - Reflected XSS
   - Stored XSS
   - DOM-based XSS
   - Cross-Site Scripting mitigation
8. **Cross-Site Request Forgery (CSRF)**
   - CSRF attacks
9. **Server-Side Request Forgery (SSRF)**
   - SSRF attacks
10. **Client-Side**
    - Bypass front-end restrictions
    - Client-side filtering
    - HTML tampering
11. **Cryptography**
    - Crypto basics
    - Encoding
    - Signing
12. **Challenges**
    - Admin password reset
    - Without account
    - Without password
