# DVWA Vulnerability Modules (from GitHub master)

1. api - API Security
2. authbypass - Authentication Bypass
3. bac - Broken Access Control
4. brute - Brute Force
5. captcha - Insecure CAPTCHA
6. cryptography - Weak Cryptography
7. csp - Content Security Policy Bypass
8. csrf - Cross-Site Request Forgery
9. exec - Command Injection (OS Command Execution)
10. fi - File Inclusion (LFI/RFI)
11. javascript - JavaScript Attacks (Client-side)
12. open_redirect - Open Redirect
13. sqli - SQL Injection
14. sqli_blind - Blind SQL Injection
15. upload - File Upload
16. weak_id - Weak Session IDs
17. xss_d - DOM-based XSS
18. xss_r - Reflected XSS
19. xss_s - Stored XSS

Each module has 4 security levels: Low, Medium, High, Impossible
